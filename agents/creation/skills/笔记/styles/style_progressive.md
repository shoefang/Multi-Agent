## Design Aesthetic

渐进式信息图风格 - 手机UI幻灯片，专注于**无障碍设计（Cognitive Accessibility）**与**生成式UI**美学。

设计优先级：**瞬间清晰、零认知摩擦**，同时保持**温暖、平易近人的视觉基调**。

### 核心设计哲学

**"高信噪比" + "温度感"**
- **高信噪比**：尽量精简文案，将复杂的长句改写为简洁易懂的短句，但要保证完整描述要讲的事情。
- **温度感**：手绘贴纸风插画、柔和色彩、荧光笔高光，营造温暖平易的氛围

## Background - 语义化自适应配色

必须根据**内容主题**正确选择背景配色：

| 主题类型 | 配色 | 典型关键词 |
|---------|------|-----------|
| **生活/饮食** | 暖米色 (#FFF8E7) | 美食、饮品、奶茶、咖啡、茶、餐厅、菜谱、生活方式 |
| **科技/效率** | 冷雾蓝 (#F0F4F8) | 软件、App、数据、代码、算法、系统架构、效率工具 |
| **健康/自然** | 柔和薄荷绿 (#F0F8F4) | 运动、健身、环保、自然、健康、医疗、疗愈 |
| **文化/历史** | 奶油米白 (#F5F0E6) | 历史、传统文化、古籍、文物、怀旧、艺术 |

**判断优先级**：
1. **优先判断内容主题**，而非表现形式
2. 即使是"对比"、"评测"，如果主题是饮食/生活，仍用**暖米色**
3. 只有明确涉及技术/数据/软件时，才用**冷雾蓝**

**示例**：
- ✅ "奶茶对比测评" → 暖米色（主题是饮食）
- ✅ "App性能对比" → 冷雾蓝（主题是科技）
- ✅ "咖啡制作教程" → 暖米色（主题是饮食）

### Texture

- **微妙纸张纹理**，温暖怀旧感，类似复古印刷品
- 或**纯净数字纹理**，现代简洁感（科技主题）


### Highlighter Effect (高光强调)

模拟**荧光笔效果**（Fluorescent Highlighter Effect）标注关键步骤或配料：
- 使用**与背景形成互补的颜色**：
  - 暖底色（暖米色）配**亮橙**（#FFB347）
  - 冷底色（冷雾蓝）配**荧光黄**（#FFE066）
  - 绿底色（柔和薄荷绿）配**亮黄绿**（#CCFF00）
- 半透明背景块（70% opacity）
- 覆盖关键词和重要信息
- 模拟真实荧光笔标记

## Visual Elements - 贴纸感插画

### 图像风格

**"手绘贴纸风插画" (Hand-drawn Sticker-style)** 或 **"高品质Emoji风格图标"**

- **风格**：增加温暖感并降低阅读压力
- **限制**：
  - **避免**任何写实照片风格
  - 涉及具体人物时，请使用**极简线条**或**卡通头像**替代
- **特点**：
  - 手绘感，不完美但温暖
  - 边缘有细微的不规则感
  - 色彩饱和度适中
  - 可以有轻微的阴影或高光，增加立体感

### 核心特征：诠释力插画

**插画即信息**
- 插画不是装饰，是文本信息的辅助载体
- 通过插画讲述故事、传达概念
- 使用视觉隐喻解释抽象概念

**视觉叙事**
- 人物插画展示状态、动作、情绪
- 环境插画提供背景、氛围、时代感
- 物件插画代表概念、类别、属性
- 对比插画展示差异、关系、变化

**形式多样性**
- 列表式插画（图标+文字，竖向或横向排列）
- 场景式插画（完整场景，适应不同比例）
- 对比式插画（上下或左右对比）
- 时间轴插画（垂直或水平时间轴）
- 步骤流（箭头连接，流动展示）
- Split卡片（左右或上下对比）


## Style Rules

### 应该遵循下列要求

- **使用清晰的黑体/Sans-serif字体（强制要求）**
- **严禁手写体或书法体（强制要求）**
- 标题醒目加粗巨大
- 正文保持极高对比度（WCAG标准）
- 使用荧光笔效果高亮关键内容
- 使用手绘贴纸风插画
- 使用视觉隐喻
- 语义化配色

### 禁止下列行为

- **使用渐变或写实阴影**（保持扁平感）
- 创建逼真元素（使用贴纸风替代）
- 使用细线或变化线重
- 包含复杂细节插图（简化为贴纸风）
- **使用手写体或书法体（严格禁止）**
- **使用纯黑色文字**（使用柔和深色）
- 使用写实照片风格（使用贴纸风/Emoji风格）
- **装饰元素喧宾夺主**（保持高信噪比）


## Dimension Mapping

**Dimensions**: clean + vibrant + geometric + balanced

- **Texture**: clean (纯净边缘) OR organic (手绘贴纸感)
- **Mood**: vibrant (温暖、活泼的色调) - **语义化自适应**
- **Typography**: geometric (现代几何无衬线体) - **无障碍友好**


## 设计哲学

**"瞬间清晰 + 零认知摩擦 + 温暖平易"**

核心是为有阅读障碍的用户设计高可访问性的幻灯片式信息图：
- 插画温暖（手绘贴纸风，降低阅读压力）
- 配色语义化（根据主题自适应）
- 保持扁平化矢量风格，但增加手绘温度感
