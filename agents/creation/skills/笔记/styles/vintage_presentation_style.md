# vintage

复古做旧纸张美学风格，适用于历史主题和探险叙事类演示文稿

## Design Aesthetic

怀旧复古风格，结合做旧纸张纹理与历史文档视觉语言。  
整体感觉类似探险家日志、古老地图或博物馆展品。  
使用温暖浓郁的色调与风化质感，营造**探索、传承与永恒知识**的氛围。

## Background

- 颜色：做旧羊皮纸色（Aged Parchment #F5E6D3）或棕褐奶油色（Sepia Cream #FFF8DC）
- 纹理：明显的做旧纸张纹理，可包含轻微折痕、咖啡渍和磨损边缘

## Typography

### Primary Font（标题字体）

经典衬线字体，具有历史感（如 Garamond、Baskerville 或类似字体）。  
风格优雅、权威、具有时间沉淀感，可适当加入装饰性细节。

### Secondary Font（标注字体）

紧缩衬线字体或干净的无衬线字体，用于地图标注和说明。  
保持符合时代感，与复古风格整体一致。

### Body Font（正文）

适合长文本阅读的衬线字体。  
传统书籍排版风格，阅读舒适自然。

## Visual Elements

- 古董地图（带路线线条与地标）
- 指南针玫瑰与航海元素
- 探险船或探险交通工具插画
- 标本绘图（植物、动物、化石）
- 手写风格注释
- 绳索、皮革、黄铜等装饰元素
- 海浪与地形纹理图案
- 复古照片风格的图片边框

## Style Rules

### 应该遵循

- 在整个画面中保持一致的做旧纹理
- 使用符合时代感的视觉语言
- 在合适的地方加入地图或旅程元素
- 采用层叠拼贴式构图
- 保持温暖的棕褐色（sepia）色调体系

### 禁止行为

- 使用现代数字化设计风格
- 创建过于锐利或干净的边缘
- 使用冷色或高饱和亮色
- 添加现代元素
- 添加幻灯片编号、页脚或品牌 Logo

## Best For

历史主题演示、旅行与探险内容、博物馆展示、  
品牌历史故事、人物传记演示、科学发现叙事、历史教育内容
